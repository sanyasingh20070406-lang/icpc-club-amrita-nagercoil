Curriculum

The Ultimate Algorithmic Roadmap

Follow our structured study guide from absolute beginner to regional competitor. Master logic, conquer data structures, and crush time limits.

Phase 01 • Weeks 1-2

Syntax, Time Complexity & Basic Math

Get comfortable with C++ STL or Java Collections. Learn how to estimate if your code will pass the 1-second time limit (Big O notation) and master prime sieves and modular arithmetic.

View STL Guide
Phase 02 • Weeks 3-4

Binary Search & Two Pointers

Move beyond simple linear scans. Learn to find optimal solutions in logarithmic time using Binary Search on Answers, and optimize array traversals with Two Pointer and Sliding Window techniques.

Binary Search Practice
Phase 03 • Weeks 5-6

Graphs, DFS & BFS

Understand how to represent networks using adjacency lists. Traverse grids, find shortest paths in unweighted graphs using BFS, and explore connected components with DFS.

Graph Algorithms
Phase 04 • Weeks 7-8

Dynamic Programming (1D & 2D)

The most feared but rewarding topic. Learn how to break down complex problems into overlapping subproblems, cache results (memoization), and solve them iteratively (tabulation).

DP Roadmap
Phase 05 • Advanced Topics

Trees, Segment Trees & DSU

Master Disjoint Set Union (DSU) for cycle detection, calculate Lowest Common Ancestors (LCA) on trees, and utilize Segment Trees for blazing-fast range queries and updates.

Advanced Data Structures
